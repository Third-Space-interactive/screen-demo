# Sound Effects

Place your sound effect files here. The SoundEffects component expects:

- `click-1.mp3`, `click-2.mp3`, `click-3.mp3` - Mouse click variations
- `whoosh-in-1.mp3`, `whoosh-in-2.mp3` - Zoom-in camera whooshes
- `whoosh-out-1.mp3`, `whoosh-out-2.mp3` - Zoom-out sounds (tip: reverse your whoosh-in files)
- `whoosh-pan.mp3` - Camera pan between segments

Sound effects are optional. Set `showSfx={false}` in Root.tsx to disable.

Recommended sources for royalty-free SFX:
- Freesound.org
- Mixkit.co
- Pixabay Sound Effects
